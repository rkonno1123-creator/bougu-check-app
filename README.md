# 防護具照合チェックアプリ

建設現場における防護具（防護服・手袋・吸収缶）の使用数を、チェックシート（PDF）とExcel集計表で照合するためのWebアプリケーションです。

## 機能

- 複数業者（1〜5社）のPDFチェックシートをアップロード
- Excel集計表から日付と使用数を自動読み取り
- 日付ごとに3社分のPDFを横並び表示
- 人間が数値を入力 → 自動で合計・照合
- 結果をPDFとして出力

## セットアップ

### 必要環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
# リポジトリをクローン（またはzipを解凍）
cd bougu-check-app

# 依存パッケージをインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:3000 を開く

### Vercelへのデプロイ

```bash
# Vercel CLIをインストール（未インストールの場合）
npm install -g vercel

# デプロイ
vercel
```

または、GitHubにプッシュしてVercelダッシュボードから連携

## 使い方

### 1. 初期設定画面

1. 業者数を選択（1〜5社）
2. 各業者の名前を入力
3. 各業者のPDFチェックシートをアップロード
4. Excel集計表をアップロード
5. 「読み込み開始」をクリック

### 2. 一覧画面

- 日付ごとの照合状態を確認
- 行をクリックして詳細画面へ
- 「PDF出力」で結果をダウンロード

### 3. 詳細画面

- 3社分のPDFが横並びで表示
- 各項目（防護服・手袋・吸収缶）の数を入力
- 自動で合計計算・Excelと照合
- 「保存して一覧へ」で保存

## 対応フォーマット

### PDFチェックシート

- 1ページ = 1週間（月〜金）
- 各作業員ごとに防護服・手袋・吸収缶の行
- 日付は印字（自動読み取り）

### Excel集計表

- シート名: 「防護服」「防護手袋」「フィルター」
- 7行目以降がデータ
- 列A: 日付、列D: 使用数
